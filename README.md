# 🎮 Hangman Game (Python)

This is a simple text-based Hangman Game built using Python.  
The player has to guess the hidden word one letter at a time within limited attempts.

---

## 🚀 Features
- Random word selection
- User input for guessing letters
- Maximum 6 wrong attempts allowed
- Displays progress of guessed word
- Simple and beginner-friendly

---

## 🛠️ Technologies Used
- Python
- Random module
- Basic loops and conditions

---

## ▶️ How to Run

1. Install Python  
2. Download or clone this repository  
3. Open terminal in project folder  
4. Run the following command:

```bash
python hangman.py
